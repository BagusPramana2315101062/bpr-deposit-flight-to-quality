# Dataset

Dataset utama tidak disimpan langsung di repository ini.

## Sumber Dataset

Dataset yang digunakan adalah:

**Dataset for deposit rural banks in Indonesia**

Dataset dapat diunduh melalui Mendeley Data:

https://data.mendeley.com/datasets/ztxbdzhdws

DOI dataset:

`10.17632/ztxbdzhdws.2`

## Karakteristik Dataset

- Periode: 9 Agustus 2021 sampai 26 Agustus 2024
- Frekuensi: Mingguan
- Jumlah observasi bersih: 608
- Tipe deposan:
  - BPR
  - HNWI
  - Corporate
  - Retail

## Struktur File Excel

File Excel terdiri dari tiga sheet:

1. `Trend deposit`
2. `Type depositor`
3. `Registered User & Deposan`

Analisis utama dalam proyek ini menggunakan sheet `Type depositor`.

## Cara Menggunakan Dataset

1. Unduh dataset dari sumber resmi.
2. Buka notebook proyek menggunakan Google Colab.
3. Jalankan cell upload file.
4. Pilih file Excel dataset.
5. Pastikan nama file yang digunakan sesuai dengan nama file pada kode notebook.

## Catatan

Dataset digunakan untuk keperluan akademik. Hak penggunaan dan lisensi dataset mengikuti ketentuan yang tercantum pada halaman Mendeley Data.
